## The Problem

Building AI applications requires LLMs, vector databases, experiment tracking, observability, and deployment pipelines. Setting this up yourself means weeks of configuration, integration headaches, and ongoing maintenance. Cloud APIs mean costs that scale with usage and data leaving your network.

## The Solution

Thinkube provides a complete AI development environment that installs in hours. One installer sets up everything - from local LLM inference to GitOps deployment - configured to work together out of the box.

## Development Tools

### Thinky - AI Assistant in JupyterLab
Chat with Claude in a sidebar while working on notebooks. Thinky can read, write, and execute your cells - and iterate until they work. When code fails, Thinky sees the error, fixes it, and re-runs automatically. An autonomous coding assistant embedded in your notebook environment.

### Code Server with Claude Code
VS Code in the browser with Claude Code CLI pre-installed. Full agentic coding for all your projects, accessible from anywhere.

### Thinkube Control
Central dashboard for your entire platform. Monitor all services with health checks and real-time GPU metrics. Deploy applications from templates with one click - just pick a template, give it a name, and watch it deploy. Manage your container registry, AI models in MLflow, secrets, and optional components. Evolving into an MCP server for LLM-based platform management.

## What's Included

### AI & Agents
- **LLM Inference**: Ollama, vLLM, TensorRT-LLM, LiteLLM
- **Vector Databases**: Qdrant, Weaviate, Chroma
- **Observability**: Langfuse for LLM monitoring
- **ML Platform**: JupyterHub + Thinky, MLflow

### Development
- **Code Editor**: Code Server + Claude Code
- **Git Server**: Gitea with CI/CD integration
- **Container Registry**: Harbor

### Infrastructure
- **Databases**: PostgreSQL, Valkey (Redis), ClickHouse
- **CI/CD**: Argo Workflows, ArgoCD
- **Monitoring**: Prometheus, Perses
