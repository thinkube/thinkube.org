## Key Features

### 🚀 Simple Deployment
Define your application in a simple `thinkube.yaml` file:
```yaml
name: my-app
type: webapp
image: myapp:latest
domain: app.example.com
```

### 🔒 Security Built-in
- Automatic TLS certificates
- Network policies by default
- Secret management with Vault
- RBAC and OAuth2/OIDC support

### 📊 Complete Observability
- Prometheus metrics
- Grafana dashboards
- Centralized logging
- Distributed tracing

### 🔄 GitOps Workflow
- Git as single source of truth
- Automatic deployments on commit
- Easy rollbacks
- Environment promotion
