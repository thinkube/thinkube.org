# Welcome to Thinkube

![Thinkube Logo](/logo.svg)

A home-based development platform built on Kubernetes, designed to simplify deployment without requiring Kubernetes expertise.

## Key Features

- **No Kubernetes Knowledge Required**: Deploy applications using simple YAML descriptors
- **Integrated Development Environment**: Built-in code-server for web-based development
- **AI-Ready Components**: Pre-configured services for AI and machine learning workloads
- **GitOps Workflow**: Automated CI/CD with Gitea and ArgoCD
- **Production-Ready**: TLS certificates, monitoring, and logging out of the box

## Why Thinkube?

### Simplicity First
Deploy complex applications without understanding Kubernetes internals. Just describe what you want in a simple YAML file.

### Developer Experience
Integrated code-server, hot reloading, and Git integration make development seamless.

### Production Ready
Built-in TLS, monitoring, logging, and backup solutions mean your applications are production-ready from day one.

### AI Native
Pre-configured with GPU support, Jupyter notebooks, and ML frameworks for AI development.
