# thinkube.bak — pre-Antora documentation (reference only)

Snapshot of the previous Astro/Starlight + Docusaurus documentation, kept as the
**reference source for the content migration (TEP-tghabr)** — the real content
and the 35 product screenshots to mine, verify against the current product, and
reuse.

- **Not** built by Antora; **not** included in the deployed image (see `.dockerignore`).
- The full git history of the old site is also preserved at tag `pre-antora-content`.

This directory is reference material, not live docs. The current repo is the new
Antora platform; real content is migrated *onto* it from here in TEP-tghabr.
