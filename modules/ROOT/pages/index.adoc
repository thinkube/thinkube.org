== The four page-types

Every page is exactly one type, each with its own visible treatment:

* xref:tutorial.adoc[Tutorial] — learning by doing
* xref:how-to.adoc[How-to] — solve one task
* xref:reference.adoc[Reference] — look it up
* xref:concept.adoc[Concept] — understand it

NOTE: Teal, Poppins, sharp corners, and the DGX Spark stamp in the corner — if you
see those, the branded build works.
