= What Is Thinkube
:page-type: concept

Understanding-oriented explanation — the *concept* page-type. It builds the mental
model, with no steps to follow.

Thinkube is a single-user, self-hosted platform for the whole AI/ML development
lifecycle — get a model, fine-tune it, build apps that consume it, and serve it —
on your own hardware, with the power of a CNCF-conformant Kubernetes and none of
the Kubernetes knowledge required.

The loop it closes, rendered as a d2 diagram (proof of the Kroki/d2 path):

[d2]
----
direction: right
models: Get models
finetune: Fine-tune
build: Build apps
serve: Serve

models -> finetune -> build -> serve
serve -> build: iterate {style.stroke-dash: 3}
----
