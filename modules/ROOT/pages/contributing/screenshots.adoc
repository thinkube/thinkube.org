== Capture plan (the follow-on spec)

🤖 *Playwright set* — log in once through Keycloak (`auth.thinkube.com`) in the
Playwright MCP browser, then for each 🤖 id navigate its URL and screenshot to
`<id>.png`, replacing `<id>.svg` in the owning component's `modules/ROOT/images/`.

✋ *Manual set (yours)* — capture each from your logged-in session and drop the file
in by id: the cockpit sessions, a live Thinky notebook, a fine-tune mid-run, an
`nvidia-smi` terminal, and the **full current installer walkthrough** (Tauri desktop
app), replacing the stale set in `thinkube-installer`.
