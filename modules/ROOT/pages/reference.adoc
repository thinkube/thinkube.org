= thinkube.yaml Reference
:page-type: reference

Lookup material — the *reference* page-type. Dry, complete, scannable.

The example below is *not* hand-typed — it is `include::`-d from the real
`thinkube.yaml` that deploys this very documentation site, so it can never
silently drift from a working manifest:

[source,yaml]
----
include::example$thinkube.yaml[]
----

[cols="1,1,3"]
|===
| Field | Type | Description

| `deployment.type` | string | `app`, `knative`, or `component`.
| `containers[].build` | string | Build context (a path, e.g. `.`).
| `containers[].port` | integer | Port the container listens on.
| `containers[].size` | string | `small`, `medium`, `large`, `xlarge`.
| `containers[].health` | string | HTTP path probed for readiness.
| `routes[].path` | string | Path prefix routed to a container.
| `routes[].to` | string | Name of the container the path routes to.
|===
